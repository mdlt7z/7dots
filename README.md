# 7dots

A little expression with seven dots :)

7dots is a watch face that lets you get information minimally with seven dots.

## Dot Functions

- 📅 Day of the week
- 👟 Step goal progress
- ☁️ Weather forecast
- ☀️ Sunrise/Sunset
- 🔋 Battery

## Compatible Models

Currently, only color models are supported.

- Basalt
- Chalk
- Emery
- Gabbro
