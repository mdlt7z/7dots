# 7dots

7dots is a watch face that lets you get information minimally with seven dots.

![screenshot_basalt](misc/screenshot_basalt.png)

## Dot Features

- 📅 Day of the week
- 👟 Step goal progress
- ☁️ Weekly weather forecast
- ☀️ Sunrise/Sunset
- 🔋 Battery

## Compatible Models

Currently, only color models are supported.

- Basalt
- Chalk
- Emery
- Gabbro
